# Физический калькулятор: законы Ньютона и механика (Вариант 3)

Курсовой проект по веб-разработке. Представляет собой веб-приложение для автоматизированного решения задач по классической механике (кинематика, динамика, работа и энергия, импульс, гидростатика). 

Система умеет динамически вычислять неизвестные параметры на основе введенных данных, строить пошаговое математическое решение с выводом красивых LaTeX-формул и визуализировать базовые физические схемы на Canvas.

## 🚀 Используемые технологии
* **Frontend:** HTML5, CSS3 (Glassmorphism design), JavaScript (ES6+), Fetch API, MathJax (для рендера формул), Canvas API.
* **Backend:** PHP 8.x, PDO, объектно-ориентированная архитектура вычислений (написание собственного парсера выражений вместо `eval()`).
* **База данных:** MySQL / MariaDB. JSON-структуры для хранения конфигураций форм и EAV-моделей задач.

## 📋 Основной функционал
* **Динамический калькулятор:** Поля ввода генерируются на лету в зависимости от выбранного типа задачи (парсинг из БД).
* **Пошаговое решение:** Алгоритм не просто выдает ответ, а расписывает подстановку каждого значения.
* **Конструктор задач:** Пользователи могут создавать собственные задачи, фиксируя часть переменных как константы.
* **Личный кабинет:** История вычислений с возможностью пагинации, удаления и повторного использования (в один клик).
* **Админ-панель (CRUD):** Полное управление пользователями, справочником констант и типами задач (сбор статистики использования).

## ⚙️ Инструкция по установке (OpenServer)
1. Скопируйте папку с проектом в директорию `domains` вашего OpenServer (например, `domains/physcalc.local`).
2. Создайте новую базу данных в phpMyAdmin `coursework_bd`.
3. Импортируйте файл базы данных: выполните SQL-скрипт из файла `sql/database.sql`.
4. Настройте подключение к БД: откройте файл `includes/db.php` и при необходимости измените логин (`root`), пароль (`root` или пусто) и имя базы данных.
5. Перезапустите OpenServer и откройте проект в браузере.
